En este proyecto se llevó a cabo la realizacion de un pagina web e-comerce.

Para esto se utilizó REACT, para la realizacion del front end. En este se desarrollaron varias pantallas, 'Home' en donde se encuentran todos los productos, 'Contact' en donde la persona podría comunicarse con el dueño de la pagina, 'Cart' para poder visualizar el carrito de compras, 'login' y 'register' para registrarse y loguearse en el sistema, y finalmente 'productDetail' para poder ver el producto seleccionado. Tambien se usó un router para direccionar las rutas de la pagina, y se usó CSS para darle el diseño a la pagina.

Para la realizacion del backend, se utilizó NODE.js, JWT para la autenticación, SQL como la base de datos y .env para la proteccion de los datos sensibles.

Se utilizó Clever Cloud para la base de datos, la cual utiliza XAMP.
Se utilizó Render para poner el hosteo gratuito del servidor.
Se utilizó vercel para el deploy del frontEnd.
